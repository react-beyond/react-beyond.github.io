# deepTooltipProp

Provides a `_tooltip` prop which will be fed into a render function to render a tooltip.

## Installation

```bash
npm install react-beyond @react-beyond/tooltipprop
```

## Usage

```
deepTooltipProp(opts: {
  id?: string
  render: ({ trigger, propValue }) => ReactElement
} = {}): HOC
```

- `id` - The id of the HOC. Defaults to `"tooltipProp"`.
- `render` - A function that takes an object with the following properties:
  - `trigger` - This is the base element without the `_tooltip` prop.
  - `propValue` - The value of the `_tooltip` prop.

(Check out the [Usage Patterns](/docs/guide/usage-patterns) to see alternative syntaxes.)

## Examples

```tsx
import { Beyond } from "react-beyond"
import { deepTooltipProp } from "@react-beyond/tooltipprop"

import Tooltip from '@mui/material/Tooltip'

const el = (
  <Beyond hocs={[
    deepTooltipProp({
      render: ({ trigger, propValue }) => <Tooltip title={propValue}>{trigger}</Tooltip>
    })
  ]}>
    <App />
  </Beyond>
)
```

Elements like this under `<App />`:

```tsx
<button
  onClick={() => setCount((count) => count + 1)}
  _tooltip="Hey!"
>
  count is {count}
</button>
```
