# deepIfProp

Conditionally renders elements with an `_if` prop.

## Installation

```bash
npm install react-beyond @react-beyond/ifprop
```

## Usage

```
deepClsxProp(opts: {
  id?: string
} = {}): HOC
```

- `id` - The id of the HOC. Defaults to `"ifProp"`.

(Check out the [Usage Patterns](/docs/guide/usage-patterns) to see alternative syntaxes.)

## Examples

```tsx
import { Beyond } from "react-beyond"
import { deepClsxProp } from "@react-beyond/ifprop"

const el = (
  <Beyond hocs={[deepIfProp()]}>
    <App />
  </Beyond>
)
```

Elements like this under `<App />`:

```tsx
<div _if={true} className="some-class" />
<div _if={false} className="other-class" />
```

will turn into this:

```tsx
<div className="some-class" />
<></>
```
