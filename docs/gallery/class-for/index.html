deepClassForProp

Transforms all "class" and "for" props to "className" and "htmlFor".

import "./styles.css";

document.getElementById("app").innerHTML = `
<h1>Hello world</h1>
`;
Open Sandbox

Docusaurus green and Facebook blue are my favorite colors.

I can write Markdown alongside my JSX!

Installation

npm install react-beyond @react-beyond/classforprop

Usage

deepClassForProp(opts: {
  id?: string
  useClassOnComponents?: boolean
} = {}): HOC
  • id - The id of the HOC. Defaults to "classForProp".
  • useClassOnComponents - Whether to replace the class prop on components too, not just on DOM elements. It never replaces the for prop on components. Defaults to false.

(Check out the Usage Patterns to see alternative syntaxes.)

Examples

import { Beyond } from "react-beyond"
import { deepClassForProp } from "@react-beyond/classforprop"

const el = (
  <Beyond hocs={[deepClassForProp()]}>
    <App />
  </Beyond>
)

Elements like this under <App />:

<div class="some-class" />

will turn into this:

<div className="some-class" />
